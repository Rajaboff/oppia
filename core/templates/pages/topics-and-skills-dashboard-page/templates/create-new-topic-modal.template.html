Создать тему

Название
Название темы должно быть не более <[MAX_CHARS_IN_TOPIC_NAME]> символов.
Тема с таким названием уже существует.
Фрагмент URL темы
Фрагмент URL-адреса темы используется для однозначного доступа к странице просмотра темы. Он должен состоять из одного или нескольких слов, разделенных дефисом, в нижнем регистре., максимум с <[MAX_CHARS_IN_TOPIC_URL_FRAGMENT]> символов. Пожалуйста, используйте ключевые слова, и избегайте таких слов, как "and", "of", или "the". This topic can be accessed at the following URL:
<[hostname]>/learn/staging/<[newlyCreatedTopic.urlFragment]>
Фрагмент URL этой темы уже существует.
Описание
Описание темы должно быть не более <[MAX_CHARS_IN_TOPIC_DESCRIPTION]> символов.
